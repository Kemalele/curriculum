## Number

Можно использовать все математические операции.
Имеется встроенная библиотека `Math`.

```js
2 + 2 // сложение
3 - 3 // вычитание
10 / 2 // деление
42 * 1 // умножение
10 % 2 // остаток от деления

2.3 + 2.7 // операции с числа с плавающей точкой

console.log(Math.PI) //  вывод числа Пи
```
