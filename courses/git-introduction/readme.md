# Основы Git 
Git - один из самых популярных инструментов в арсенале программиста. Все IT организации используют git или поднобный инструмент.  
В этом курсе ты изучишь основы git, команды такие как:  
- git add   
- git commit   
- git push   
- git clone   
- git pull   
- git branch   
- git reset   
- git log   
- git merge
